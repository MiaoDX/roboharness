Why This Matters

The report is not just for humans to admire. It is the compact handoff format that lets a coding agent stop guessing from logs and start editing the right part of the code.

Rough sketch only. System fonts, gray borders, no polish on purpose. The real design work is deciding what evidence the agent sees first and what human glue work disappears.
